## 📌 Overview

**Python Utility Toolkit** is an interactive command-line application designed to provide a collection of useful utilities through a simple and efficient console interface.

The application combines multiple functionalities, including:

- 🔑 Key-value data storage
- 💾 File saving and loading
- ✏️ Text manipulation
- 📊 Word counting utilities

The project focuses on applying core Python programming concepts such as data structures, file handling, string processing, and user interaction.

---

## 🚀 Features

### 🔑 Key-Value Storage

A dictionary-based storage system for managing information dynamically.

Supported commands:

- `SET` — creates and stores a new key-value pair
- `GET` — retrieves stored information by key
- `SAVE` — exports current data into a file
- `LOAD` — restores saved data from a file

---

### ✏️ Text Processing

Built-in tools for working with text:

- `REVERSE` — reverses user-provided text
- `COUNT-WORDS` — analyzes a text file and counts total words
